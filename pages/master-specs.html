How to Use This Data

01 Start with your use case.

If you need off-grid solar, focus on the Power Stations table. If you're troubleshooting a Prius hybrid battery, the Hybrid & EV Battery table is your starting point.

02 Cross-check cell chemistry.

LiFePO4 and LFP are the same chemistry. Li-ion is a different formula with a different cycle-life profile. Don't mix these up when planning a system or shopping for a replacement.

03 Read the error codes.

If a device in this table is throwing a code you can find the meaning and first steps to diagnose it.

04 Verify against the manual.

Specs can change mid-production. Always confirm a spec against the manufacturer's manual for your exact hardware revision and firmware.
