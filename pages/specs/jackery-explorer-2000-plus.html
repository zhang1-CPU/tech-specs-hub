Jackery Error Code Reference

The complete list of error codes Jackery's manual doesn't fully document, cross-referenced from multiple field reports and BMS diagnostics.

Code Severity Description / Trigger Action
ERR1 Warning Low Battery Warning Battery level below ~10%. Stop discharge and recharge immediately. Do not deep-discharge below ~5% — accelerates cell degradation.
ERR2 Critical AC Overload Disconnect loads exceeding 3,000W continuous rating. Let unit cool for 2 minutes. Verify total connected wattage before retrying.
ERR3 Warning Over-temperature Fans blocked or ambient temperature above 40°C. Move unit to shaded, ventilated area. Remove dust/debris from intake vents. Allow 10-15 min to cool.
ERR4 Critical Internal Fault BMS or inverter communication error. Perform full reset: power off, disconnect all loads, hold power button for 15 seconds. If persists, warranty service required.
ERR5 Warning Charger Fault Wrong AC adapter detected, faulty charger cable, or intermittent wall voltage. Use only original Jackery 1,800W charger. Test with a known-good wall outlet.
ERR6 Warning DC Output Overload DC car port or 12V rail overloaded. Disconnect device from DC output. Check device fuse and current rating. Max 10A / 120W from DC car port.
ERR7 Critical Fan Fault Cooling fan stalled or blocked. Check for physical obstruction. Do not operate without cooling — thermal runaway risk. Warranty service typically required.

Troubleshooting Guide

The most common user-reported issues and practical diagnostic steps before contacting Jackery support.

Unit Won't Turn On

  1. Verify battery state of charge — if below 10%, connect to AC first.
  2. Hard reset: Disconnect all loads, then hold the main power button for 10 continuous seconds.
  3. Check that the LCD display responds — if completely dark, verify fuse on AC input board.
  4. Connect charger and observe if ERR code appears on the display.
  5. If still unresponsive, the main BMS relay may have failed — warranty claim.

Won't Charge From Solar

  1. Minimum open-circuit voltage: 25V from the panel array.
  2. Check panel polarity — reverse connection silently fails, no error code displayed.
  3. Verify MC4 connectors are fully clicked into place and not corroded.
  4. Shade on part of array can trigger MPPT false-detection (see Known Issues below).
  5. Check panel VOC in full sun with multimeter. Should exceed 32V for typical 36-cell panels.

AC Outlet Not Working

  1. AC mode requires an explicit press of the AC button on the front panel — it does not auto-activate.
  2. Check if ERR2 (overload) or ERR4 (internal) is displayed.
  3. Test with a simple, known-good resistive load (lamp or space heater) before assuming the inverter has failed.
  4. If an inductive load (fridge, pump) was connected, the startup surge may have tripped inverter protection.

Screen Shows Gibberish / Corrupted UI

  1. Firmware bug in early production units. Perform firmware update via the Jackery mobile app first.
  2. Ensure Bluetooth connection is stable during the update — do not close the app.
  3. If display is frozen, perform a 15-second button-hold reset and re-apply AC power.
  4. Persistent ghosting or vertical lines may indicate hardware display failure — warranty replacement.

Known Issues & Firmware Notes

Hardware and software issues documented from real user reports. Jackery typically addresses these under warranty.

Loose BMS Connector — Early Production Units

A documented connector issue in the first production run (approximately serial numbers before JP-250400000). Symptoms: intermittent power loss, inconsistent battery level reporting, and occasional ERR4 display.

Jackery has acknowledged this and will replace units under warranty. If you purchased in the first 90 days of launch, confirm the revision with Jackery support.

Solar MPPT False-Detection in Low-Light Conditions

Firmware v1.0.0.4 release notes acknowledge that the MPPT controller can falsely detect panel voltage and fail to start charging in dawn/dusk or heavily-overcast conditions. The controller may also oscillate between detecting and not-detecting at panel VOC near the 25V threshold.

Update to the latest available firmware via the Jackery mobile app. If issue persists, test with a higher-voltage panel string (2s configuration, 60-72V VOC) for more reliable detection.

LCD Display Scratch Susceptibility

The front display is covered with a standard acrylic panel that scratches if wiped with a dry cloth or abrasive material. Multiple user reports of cosmetic scratching after transport without protective cover.

Apply a tempered-glass screen protector (5-inch tablet size fits approximately) before first transport. Use only microfiber cloth dampened with distilled water to clean.

How to check firmware version: Long-press the Info button (or equivalent) for 5 seconds. The firmware revision will display on the LCD. If you don't have the latest, pair with the Jackery app and navigate to Settings → Firmware Update.
