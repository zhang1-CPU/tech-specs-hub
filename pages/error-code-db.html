Hybrid & EV Battery

Common OBD-II and manufacturer-specific hybrid fault codes. Covers Toyota, Lexus, Ford, and similar systems.

9 codes
Code Description Severity Typical Cause Action
P0A7F Hybrid Battery Pack "A" Performance Warning Degraded cells causing module voltage spread beyond tolerance. Measure individual module voltages. Replace weakest modules or recondition pack.
P0A80 Replace Hybrid Battery Pack Critical Overall pack capacity has dropped below the manufacturer's replacement threshold. Plan full pack replacement. Check module-to-module variance first to confirm.
P0A7B Hybrid Battery Voltage Sensor "B" Circuit Warning Faulty voltage sensor, damaged wiring harness, or loose connector on battery module. Inspect sensor connector. Measure sensor output voltage. Replace sensor if out of spec.
P0A9D Hybrid Inverter / Converter Performance Critical Inverter IGBT failure, over-temperature shutdown, or internal short in HV converter. Check inverter coolant flow and temperature. Scan for sub-codes. Replace inverter if confirmed.
P0AC4 Hybrid Battery Pack Temperature Sensor "C" Circuit Warning Failed thermistor, damaged wiring, or corroded connector on battery pack temp sensor. Measure sensor resistance at operating temperature. Replace sensor or repair wiring.
P3000 Hybrid Battery Control System Malfunction Critical Battery ECU internal fault or corrupted calibration. Scan with manufacturer-specific tool. Re-flash ECU if advised; otherwise replace ECU.
P3004 Battery ECU Communication Lost Warning CAN bus wiring issue, loose connector, or blown fuse in HV control circuit. Check fuses, reseat ECU connectors, measure CAN-H / CAN-L resistance (should be ~60 Ω).
P3009 Hybrid Battery System Over-Temperature Critical Blocked cooling fan, failed temperature sensor, or extreme load in hot conditions. Inspect cooling fan and intake filter. Replace temp sensor if out of spec. Let pack cool before reuse.
C1259 Hybrid System Malfunction (Brake / Regen) Warning Regenerative braking coordination fault between brake ECU and hybrid ECU. Check brake fluid level. Scan for sub-codes. Clear codes and test drive to confirm.

Power Stations

EcoFlow-style error codes. Codes with similar meanings on Jackery, Bluetti, and Anker models are usually named identically or differ only by a numeric prefix.

12 codes
Code Description Severity Typical Cause Action
E0 AC Over-Temperature Critical Inverter heat sink above threshold. Heavy sustained load, blocked ventilation, or failed fan. Reduce load. Power off. Clear ventilation openings. Allow 30 min to cool. If returns, service.
E1 DC Over-Temperature Critical Battery cells or charge controller overheating. Often triggered by fast charging in hot weather. Stop charging. Move to cooler environment. Do not discharge until temperature is in-range.
E2 Battery Over-Voltage Warning Charging at too high a voltage, or a cell string that has drifted high. Stop charging. Disconnect solar/wall input. Let BMS balance. If persistent, check for failing cell.
E3 AC Overload (X-Boost) Warning Connected AC load exceeds the inverter's continuous rating for more than a few seconds. Unplug devices. Total AC wattage must be below the unit's continuous rating. Check for startup inrush.
E4 Input Over-Current Warning Solar array or adapter current exceeds the MPPT's input limit. Reduce solar array wattage. Check panel open-circuit voltage is within the MPPT's Voc range.
E5 Communication Error Info Internal bus between BMS and main MCU has lost sync, or an expansion battery is not responding. Power cycle the unit. Reseat expansion battery connectors. If returns, check for firmware update.
E6 Battery Cell Failure Critical One or more cells reporting voltage significantly outside the pack's normal range. Stop using the unit. Under-warranty: contact manufacturer for replacement. Out of warranty: service.
E7 Fan Blocked Warning Inverter cooling fan unable to spin. Dust, debris, or mechanical failure. Power off and unplug. Visually inspect fan. Blow out debris with compressed air (low pressure).
E8 Solar Input Issue Warning No solar charging detected. Open circuit, reverse polarity, or panels outside MPPT voltage range. Check panel connections. Verify Voc is within MPPT range. Check for blown inline fuse.
E9 AC Charging Issue Warning Wall charging not working. Faulty adapter, loose DC input jack, or failed charging circuit. Test wall outlet. Try different adapter/cable. Check DC input port for damage.
E10 Battery Under-Voltage Critical Pack voltage below safe threshold. Often from long-term storage without top-off charge. Attempt low-current recovery charge. If BMS won't allow, replace battery pack.
E11 Inverter No Output Critical AC outlets not producing power even though unit is on. Inverter relay or MOSFET failure. Check AC output breaker. Reset unit. If no output, service inverter module.

Drones

Most common gimbal, IMU, compass, ESC, and battery faults reported by DJI and similar drones.

12 codes
Code Description Severity Typical Cause Action
GIMBAL MOTOR STUCK Gimbal motor unable to move Warning Physical obstruction, bent gimbal arm, debris in motor, or crash damage. Power off. Inspect gimbal. Remove debris. If mechanically bent, replace gimbal assembly.
GIMBAL OVERLOAD Gimbal motor current over limit Warning Filter too heavy for gimbal rating, or gimbal hit during flight. Remove any filter/add-on and retest. Verify gimbal payload is within spec.
IMU CALIBRATION ERROR IMU calibration failed Info Drone not on a perfectly level surface, or vibration during calibration. Sometimes a failing IMU. Place drone on a known-level surface. Retry calibration. If it fails repeatedly, service IMU.
COMPASS ERROR Compass reading unreliable or outside tolerance Warning Magnetic interference (metal, power lines, motors), loose compass module, or failed calibration. Move away from metal/steel structures. Recalibrate compass in an open area. Check module connector.
ESC ERROR Electronic speed controller fault Critical Damaged motor winding, shorted ESC, prop strike, or overheating during sustained high-thrust flight. Land immediately. Inspect motor and ESC for damage. Replace matched ESC/motor pair.
BATTERY COMMUNICATION ERROR Flight controller lost battery telemetry Warning Dirty or corroded battery pins, third-party battery without proper serial interface. Clean battery contacts with isopropyl alcohol. Use only manufacturer-approved batteries.
BATTERY OVER-DISCHARGE Battery cell voltage below minimum Critical Battery left fully discharged for weeks, or deep-discharged during a long flight. Do NOT attempt to fast-charge a deeply-discharged battery. Slow-charge at 0.1C. If voltage doesn't recover, retire the battery.
BATTERY OVER-TEMPERATURE Battery temperature above limit Warning High-ambient flight, aggressive maneuvers, or direct sun on battery. Land. Let battery cool to room temperature before charging or re-flight.
VISION POSITIONING ERROR Vision positioning system unavailable Info Dirty or obstructed downward-facing camera, low-light conditions, or reflective surface below. Clean vision sensor lenses. Fly in well-lit area. Avoid flying over water or glass.
GPS SIGNAL WEAK GPS satellites not sufficient for positioning Warning Indoor flight, urban canyon, heavy cloud cover, or GPS module malfunction. Move to open area. Wait for GPS lock before takeoff. Check GPS antenna connection.
RC CONNECTION LOST Remote controller signal lost Critical Out of range, interference, low RC battery, or damaged antenna. Stay within rated range. Point antennas toward drone. Return-to-home will activate automatically.
GIMBAL CALIBRATION ERROR Gimbal calibration failed Info Drone not level during calibration, or gimbal mechanical issue. Place drone on level surface. Retry calibration. Check gimbal for physical damage.

Smart Home

Robot vacuums (Roomba, Roborock generic faults), Nest thermostats, and Ecobee sensor errors.

15 codes

Robot Vacuum — Common Errors

Code Description Severity Typical Cause Action
Error 1 Wheels stuck Info Hair/string wrapped around wheel axle, or robot trapped. Inspect wheels. Remove debris. Manually spin wheels to confirm free movement.
Error 2 Main brush jammed Info Hair, string, or carpet fibers wound around the main roller brush. Remove brush module. Clean end caps. Use the brush-cutting tool if included.
Error 3 Side brush error Info Side brush motor stalled or drawing too much current. Remove side brush. Clean hub. Re-seat. If persistent, replace side brush motor.
Error 4 Cliff sensor dirty Warning Dust or debris covering the downward-facing cliff sensors. Wipe cliff sensor lenses with a clean, dry cloth. Avoid liquids.
Error 5 Bin missing or not detected Info Dust bin not seated, or optical sensor dirty. Remove and re-insert bin. Clean bin sensor window.
Error 6 Bumper stuck Info Debris under or behind the front bumper preventing movement. Tap the bumper gently to dislodge. Clean around the bumper edges.
Error 7 Wheel encoder issue Warning Wheel encoder not registering movement. Often a dirty optical disk or broken wire. Clean encoder disk. If error persists, replace the wheel module.
Error 8 Vacuum fan / suction motor Warning Blocked airway, clogged filter, or failed suction motor. Clean filter. Check airway for obstructions. If returns with clear airway, replace motor.
Error 9 Battery fault Critical Battery outside temperature range, or aged battery failing to hold charge. Let battery reach room temperature. If error persists, replace battery.

Nest Thermostats

Code Description Severity Typical Cause Action
E74 No power to Rh wire Warning Disconnected or loose Rh wire, or fuse blown at the HVAC system. Check Rh wire connection at thermostat and at HVAC. Check HVAC fuses / breakers.
E73 No power to Rc wire Warning Disconnected or loose Rc wire, or fuse blown in cooling circuit. Check Rc wire connection. Confirm HVAC fuse is intact. Inspect for loose splice behind the wall.
E194 Heat Link / relay communication lost Warning Heat Link lost power, or Wi-Fi / Zigbee mesh between thermostat and Heat Link dropped. Confirm Heat Link has power. Re-pair thermostat to Heat Link. Check for Wi-Fi interference.
H71 Humidity sensor out of range Info Humidity sensor reporting implausibly high or low value. Usually condensation after AC run. Allow unit to ventilate. If code persists, perform a factory reset of the thermostat.
C155 C wire / common wire not detected Info Install attempted without a common wire on a system that requires one. Install a C wire, or use the Nest Power Connector / the included "C wire adapter" if your model ships with one.

Ecobee Sensors

Code Description Severity Typical Cause Action
Sensor Not Detected Room sensor lost from the sensor list Info Dead sensor battery, or sensor outside radio range of the thermostat. Replace CR2032 battery. Move sensor closer to thermostat. Re-pair in Ecobee settings.

General Smart Home

Code Description Severity Typical Cause Action
DEVICE OFFLINE Device Offline / Wi-Fi Issues Warning Wi-Fi signal too weak, router issue, wrong credentials, or device firmware problem. Check Wi-Fi signal. Restart router and device. Re-enter Wi-Fi credentials. Update firmware.

3D Printers

Common FDM 3D printer errors and print quality issues. Applicable to Creality, Prusa, Bambu Lab, AnkerMake, and similar printers.

12 codes
Code Description Severity Typical Cause Action
THERMAL RUNAWAY Thermal Runaway Protection Triggered Critical Thermistor disconnected, loose, or faulty. Heater cartridge not reaching target temp. Power off immediately. Check thermistor wiring and connection. Replace if damaged.
BED LEVEL Bed Leveling / First Layer Issues Warning Uneven bed, incorrect Z-offset, or dirty build surface. Re-level bed or run auto-bed-leveling. Adjust Z-offset. Clean build plate with isopropyl alcohol.
FILAMENT JAM Filament Jam / Clogged Nozzle Warning Filament grinding in extruder, heat creep, or debris in nozzle. Unload filament. Clear nozzle with cold pull or needle. Check extruder tension.
LAYER SHIFT Layer Shifting / Misalignment Warning Loose belt tension, stepper motor current too low, or obstruction on axes. Check belt tension. Lubricate rails/rods. Verify stepper motor current.
UNDER EXTRUSION Under-Extrusion Info Clogged nozzle, worn extruder gear, or incorrect E-steps. Clean nozzle. Check extruder gear for wear. Calibrate E-steps.
OVER EXTRUSION Over-Extrusion Info Flow rate too high, incorrect filament diameter setting, or high nozzle temp. Reduce flow rate. Verify filament diameter in slicer. Calibrate E-steps.
STRINGING Stringing / Oozing Info Retraction distance/speed too low, high temp, or travel speed too slow. Increase retraction. Lower nozzle temp slightly. Increase travel speed.
WARPING Warping / Curling Warning Bed temp too low, poor bed adhesion, or cool drafts near printer. Increase bed temp. Use brim/raft. Enclose printer or use draft shield.
Z BANDING Z-Axis Banding / Ribbing Info Z-rod bent, Z-step layer height mismatch, or loose Z-coupler. Check Z-rod straightness. Tighten couplers. Verify layer height is multiple of full step.
HEATER FAULT Nozzle Heating Failed Critical Heater cartridge burned out, broken wire, or MOSFET failure. Check heater cartridge resistance. Inspect wiring. Replace heater or MOSFET if needed.
STEPPER SKIP Stepper Motor Skipping Steps Warning Motor current too low, print speed too high, or mechanical binding. Increase stepper current. Reduce print speed. Check for binding on axes.
BLTOUCH ERROR BLTouch / ABL Sensor Error Warning BLTouch pin stuck, wiring issue, or incorrect probe offset. Check probe pin movement. Verify wiring. Re-measure and set Z-offset.

E-Bikes

Common e-bike fault codes for hub motors, mid-drives, BMS, and controllers. Applicable to Bosch, Shimano, Bafang, and similar systems.

10 codes
Code Description Severity Typical Cause Action
MOTOR HALL Motor Not Working / Hall Sensor Error Critical Failed hall sensor in motor, damaged motor wire harness, or loose connector. Check motor connector and wiring. Test hall sensor output. Replace motor if sensor is bad.
BMS COMM BMS Communication Error Warning Battery BMS not responding to controller. Loose discharge connector or BMS fault. Reseat battery connector. Check for bent pins. If persistent, test battery on charger.
THROTTLE Throttle Not Responding Warning Faulty throttle, broken wire, or out-of-range voltage signal. Check throttle connector. Test throttle output voltage. Replace throttle if defective.
PAS ERROR Pedal Assist Not Working Info Damaged PAS sensor, misaligned magnet ring, or loose cadence sensor. Check PAS sensor alignment. Inspect magnet ring for damage. Adjust sensor gap.
BATT LOW Battery Not Charging / Low Voltage Warning Faulty charger, damaged charge port, or BMS shutoff due to low cell voltage. Test charger output voltage. Check charge port for damage. Try different charger.
CTRL TEMP Controller Over-Temperature Critical Continuous high-current draw, hot weather, or blocked controller heat sink. Stop riding and let controller cool. Reduce assist level. Check for blocked ventilation.
BRAKE SENS Brake Sensor Error Info Brake lever sensor stuck or misadjusted, causing constant motor cut-off. Check brake levers return fully. Test brake sensor. Adjust or replace sensor.
DISPLAY Display / LCD Not Working Warning Loose display cable, faulty display unit, or no power from controller. Check display connector and cable. Test for voltage at display plug. Replace display.
SPEED LIM Speed Limiter Issue Info Speed sensor not reading correctly, or controller speed limit misconfigured. Check speed sensor alignment on wheel. Verify controller settings and wheel size.
WHEEL SPD Wheel Speed Sensor Error Warning Speed sensor too far from magnet, damaged wire, or missing magnet on spoke. Adjust sensor gap to 1-3mm. Check magnet is present. Inspect wiring.
