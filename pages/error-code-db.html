Hybrid & EV Battery Codes

Toyota, Lexus, Honda hybrid systems — HV battery pack, inverter, and control module diagnostics.

High-voltage hazard. Codes P0A80, P0A7F, and P0A7E indicate high-voltage battery pack faults. Always de-power the HV system and wear insulated gloves before opening the battery service cover. Measure interlock continuity and service plug resistance before proceeding.
Code Description Severity Affected Models Typical Cause Action
P0A80 Replace Hybrid Battery Pack Critical Toyota Prius 2004–2022, Camry Hybrid, Highlander HV, Lexus CT200h Pack State-of-Health (SOH) below threshold; individual module voltage under 14.0V under load Measure each module voltage with load. Replace modules below 13.8V. Battery control module relearn procedure required post-install.
P0A7F HV Battery Pack Deterioration Critical Toyota Prius 2010–2022, Lexus RX450h Module balance fault; delta between strongest and weakest module exceeds 0.8V Isolate pack. Perform manual discharge-charge cycle to rebalance, or replace degraded modules. Check cooling fan operation.
P0A7E HV Battery Module Voltage Variation Warning Toyota Prius 2016–2022, RAV4 Hybrid Excessive cell imbalance (typically > 0.3V delta between modules) Record module voltages at rest and under 10A load. Charge/discharge cycle. If delta persists, replace out-of-spec module.
P0A94 DC/DC Converter Performance Warning Toyota Prius 2010+, all Toyota/Lexus hybrids using G9200 inverter 12V system charge failure; DC/DC output out of 13.8V–14.2V target window Check 12V battery condition. Verify DC/DC output at idle. Check HV fuse and connector terminals for corrosion.
P0A78 Drive Motor A Inverter Performance Critical Toyota Prius 2004–2022, Prius Prime, Corolla Hybrid MG1 power electronics fault; IGBT module thermal runaway or phase current imbalance Inspect inverter coolant flow. Check phase resistance balance between MG1 phases. Replace G9200 inverter if out of spec.
P0A1F Battery Energy Control Module Internal Warning All Toyota THS-II hybrid platforms Internal communication error between battery ECU sub-modules Update battery ECU firmware if available. Check wiring harness continuity. Replace battery ECU if hardware fault confirmed.
P3000 Battery Control System Malfunction Warning All Toyota hybrid platforms General HV fault flag — a secondary DTC is stored Always check for sub-codes. Address the specific fault before clearing P3000. Common: P0A80 or P0A7F as root cause.
P0BD5 Honda IMA Battery Module Deterioration Warning Honda Civic Hybrid 2006–2015, Insight, Accord Hybrid Individual NiMH module capacity degradation below 75% Grid-charge pack using Honda IMA charger. Replace modules showing voltage < 1.2V/cell under load. Clear code with Honda Diagnostic System (HDS).
P0A0D Hybrid Battery Temperature Sensor A Circuit High Critical All Toyota/Lexus HV battery platforms Open circuit in thermistor circuit; pack cooling fan stalled Measure thermistor resistance at connector (should read 10kΩ at 25°C). Check fan intake filter for debris. Replace sensor or fan motor as needed.
P0BBA Hybrid Battery Current Sensor Performance Warning Toyota Prius 2010–2022, Camry Hybrid Hall-effect current sensor drift or offset error outside ±2A window Check connector for corrosion. Perform current sensor zero-point calibration using Techstream. Replace sensor if calibration fails.

Portable Power Station Codes

EcoFlow E-series codes, Jackery ERR codes, Bluetti E-series, and generic fault indicators.

ECOFLOW E0–E7 Inverter & Battery Codes

Code Description Severity Affected Models Typical Cause Action
E0 AC Over-temperature Critical Delta Pro 3, Delta 2 Max, River 2 Pro Inverter heatsink temperature > 75°C; cooling airflow blocked Shut down unit immediately. Move to well-ventilated area. Clear intake/exhaust vents. Allow 30 minutes to cool before restarting.
E1 Battery Over-temperature Warning All EcoFlow LFP models Cell temperature > 45°C during charging or discharge Pause charge/discharge. Move to cooler environment. Verify ambient temperature < 35°C. Resume when cells < 40°C.
E2 DC Over-temperature Warning Delta Pro, Delta 2 series DC-DC converter thermal warning; MPPT input stage overheating Reduce solar array wattage below nameplate. Check MPPT airflow. Allow cooling period before connecting additional DC load.
E3 AC Overload Warning All EcoFlow models Continuous output exceeds rated wattage for more than 5 seconds (X-Boost excluded) Identify and remove offending AC load. Verify total wattage < unit rated output. Press power button to reset inverter stage.
E4 Input Over-current Warning All EcoFlow models with solar input Solar panel series configuration exceeds MPPT input current rating Reconfigure panels from series to parallel. Verify total short-circuit current (Isc) × 1.25 safety factor. Check panel VOC does not exceed 60V/100V rating.
E5 Communication Error Info EcoFlow expansion batteries + BMS CAN bus communication loss between master unit and expansion battery Power-cycle all units. Re-seat Anderson connector between main and expansion pack. Check for bent pins on both ends of cable.
E6 Battery Cell Failure Critical All EcoFlow LFP battery modules Individual cell voltage below 2.5V or above 3.65V; internal short suspected DO NOT charge unit. Disconnect all loads. Contact EcoFlow warranty service. Module requires BMS + cell stack replacement.
E7 Fan Blocked Warning Delta Pro, Delta 2 Max, River 2 Pro Fan tachometer reads 0 RPM; physical obstruction or fan motor failure Inspect fan intake. Clear debris. If no obstruction found, fan motor replacement required. Run 20-minute burn-in test post-repair.

JACKERY ERR-series Fault Codes

Code Description Severity Affected Models Typical Cause Action
ERR1 Low Battery Warning Jackery Explorer 300, 500, 1000, 1500, 2000 Pro Battery discharged below minimum operating voltage Connect AC charger immediately. Do not deep-discharge below 10% state of charge. Replace if battery fails to hold charge after 4–8 hours of charging.
ERR2 Overload Warning All Jackery Explorer series Connected AC load exceeds unit continuous output rating Remove surplus AC loads. Verify combined wattage with inline power meter. Inverter auto-resets after overload condition clears.
ERR3 Over-temperature Warning All Jackery Explorer series Ambient temperature > 40°C or internal thermal protection tripped Move unit to shaded, ventilated area. Disconnect high-power loads. Wait until internal temperature drops below 45°C before resuming operation.
ERR4 Internal Fault Critical Jackery Explorer 1000+, 1500, 2000 Pro BMS-inverter communication fault or protection circuit failure Full power-cycle (30 seconds off) required. If code persists, unit requires factory service. Do not attempt DIY repair — contains sealed LFP modules.
ERR5 Charger Fault Warning All Jackery Explorer series External AC wall charger adapter failure or incorrect voltage Test charger output voltage with multimeter. Replace charger if outside 19.5V–20.5V window. Verify wall socket provides correct mains voltage.

BLUETTI E-series BMS Codes

Code Description Severity Affected Models Typical Cause Action
E101 Battery Over-discharge Warning AC200P, AC300, AC500, EP500 Pack voltage dropped below low-voltage protection threshold Connect charger at low current (5A) first. After 20 minutes switch to standard charge rate. BMS will re-engage once pack recovers.
E102 Battery Over-voltage Warning All Bluetti LFP models Charge voltage exceeds per-cell 3.65V target; charger handshake mismatch Disconnect charger. Verify charger output matches pack spec. Replace charger if fault recurs. Check individual cell voltages when possible.
E103 Communication Error Info AC300 + B300 expansion packs, AC500 RS485 bus communication failure between master and slave BMS units Verify communication cable is fully seated at both ends. Power-cycle all connected units in sequence: expansions first, then master.
E104 Inverter Overload Warning AC200P, AC300, AC500 Continuous AC output exceeds inverter nameplate rating Reduce connected load. Total AC wattage must not exceed rated output. Inverter auto-recovers after 30-second cooldown with reduced load.
E105 Inverter Over-temperature Warning All Bluetti inverter-equipped models IGBT heatsink temperature exceeds 70°C thermal cutout Clear intake/exhaust vents. Move unit to ambient < 30°C environment. Inspect cooling fans for seized bearings; replace if stalled.

GENERIC Universal Power Station Fault Indicators

Code / Indicator Description Severity Affected Models Typical Cause Action
Check Engine LED Flashing General system fault indicator Warning Anker, Goal Zero, Rockpals, other non-display models Any fault condition not mapped to specific code; BMS self-test failure Count flash pattern (short vs long blinks) and cross-reference to manual. Perform full 30-second power reset. Consult manufacturer support if pattern recurs.
Fault: BMS Internal Battery Management System internal failure Critical All portable power station brands Firmware corruption, voltage sense calibration drift, or MOSFET drive circuit failure Attempt firmware update via manufacturer app if unit still boots. If unresponsive, warranty service required. Do not attempt to re-flash without OEM tooling.
Solar Input Not Detected MPPT controller not seeing solar panel Warning Any unit with solar charging input Open circuit in panel wiring; MC4 connector not fully latched; panel VOC below MPPT minimum input threshold Check panel open-circuit voltage under sunlight. Verify MC4 connectors are locked. Ensure at least 12V (for 12V MPPT) or 30V (for high-voltage MPPT) present at input.

Drone Fault Codes (DJI)

Gimbal, battery, flight controller, and ESC fault codes for Mavic, Air, and Mini series.

Gimbal Errors

Code / Indicator Description Severity Affected Models Typical Cause Action
GIMBAL MOTOR STUCK Gimbal motor over-current / stalled Warning Mavic Pro, Mavic Air 2, Air 2S Physical obstruction in gimbal travel; shock-damage misalignment after hard landing Inspect gimbal cage for debris. Check gimbal motor winding resistance (target 3.5–4.2Ω per phase). Perform gimbal calibration through DJI Fly app. Replace motor assembly if out of spec.
GIMBAL OVERLOAD Motor torque limit exceeded Warning Mavic 2 Pro, Mavic 3, Air 2S Gimbal damper balls out-of-spec or deformed; camera/filter combination overweight Inspect gimbal damper for cracked or deformed rubber. Replace ND filter with lighter alternative. Perform gimbal IMU calibration on flat surface post-repair.
GIMBAL IMU CALIBRATION Gimbal IMU calibration required Warning All DJI models with 3-axis gimbal Gyro bias drift exceeding tolerance; typically after firmware update or temperature change > 20°C Re-calibrate gimbal IMU on perfectly flat, non-vibrating surface. Use DJI-level calibration surface. If calibration fails repeatedly, IMU module faulty.
GIMBAL PITCH MOTOR Pitch-axis motor driver fault Critical Mavic Pro, Mavic 2, Phantom 4 Pro Physical damage to pitch-axis motor winding; H-bridge driver IC failure on gimbal PCB Replace gimbal pitch motor assembly. Verify motor winding continuity < 5Ω. Check gimbal PCB for visible burn marks on driver chip. Full gimbal replacement may be most cost-effective path.
GIMBAL ROLL MOTOR Roll-axis motor driver fault Critical Mavic 3, Mavic 2 Enterprise, Inspire 2 Crash-damaged roll motor; bearing seizure; cable harness fatigue at joint Inspect roll-axis for mechanical binding. Test motor phase resistance balance. Replace damaged cable harness or motor. Factory-level repair recommended for Inspire 2 X7/X5S cameras.
GIMBAL YAW MOTOR Yaw-axis motor driver fault Critical Phantom 4 Pro, Mavic Air, FPV drone Yaw motor bearing failure or driver PCB fault; cracked solder joints from vibration fatigue Check yaw motor for free rotation without excessive play. If resistance felt, replace bearing/motor assembly. Verify ribbon cable continuity between gimbal and main board.

Battery Errors

Code / Indicator Description Severity Affected Models Typical Cause Action
BATTERY COMMUNICATION Battery-BMS communication failure Warning Mavic Mini, Mini 2, Mini 3 Pro Corroded or dirty battery-to-drone contact pads; firmware mismatch between battery BMS and flight controller Clean battery contacts with 99% isopropyl alcohol and cotton swab. Update both battery and drone firmware to latest version. Verify battery insertion lock is properly engaged.
BATTERY OVER-DISCHARGE Battery cells deeply discharged Critical All DJI Intelligent Battery models Battery left in storage > 6 months; self-discharge brought cells below 3.0V minimum safe level Attempt recondition cycle: charge at 0.1C for 2 hours, then 0.5C to 3.5V/cell. If voltage does not recover after 30 minutes at 0.1C, battery is unrecoverable. Recycle at approved facility.
BATTERY TEMPERATURE HIGH Battery temperature exceeds safe window Warning Mavic 3, Mavic 2, Air 2S (high-power draw) Direct sunlight heating; post-crash internal short suspected; aggressive sport-mode flight in > 35°C ambient Land drone immediately if in-flight. Remove battery from aircraft. Allow full 15-minute cooling period before re-insertion or charging. Do not charge battery while still warm to touch.

Flight Controller & ESC Errors

Code / Indicator Description Severity Affected Models Typical Cause Action
COMPASS ERROR Magnetometer calibration out-of-tolerance Warning All DJI GPS-equipped models Flight controller compass calibration performed near metal objects; residual magnetic field in frame after crash repair Calibrate compass away from metal, power lines, and reinforced concrete. Rotate drone through full 360° in horizontal plane, then nose-down. If calibration fails at same location, try different site.
IMU ERROR Inertial Measurement Unit fault Warning All DJI flight controller models Gyro bias drift; accelerometer offset; temperature-induced calibration shift Recalibrate IMU on perfectly level, vibration-free surface. Warm up drone for 5 minutes in current environment before calibration. If IMU reports inconsistency between dual IMUs, flight controller repair required.
ESC ERROR Electronic Speed Controller fault Critical Mavic Pro, Mavic 2, Phantom series Motor winding phase short; ESC MOSFET thermal failure; damaged motor/ESC cable from crash impact Ground drone — do not fly. Identify affected motor/ESC by error position. Test motor phase resistance balance (all three phases should match within ±5%). Replace motor + ESC pair as matched set.

Smart Home Device Codes

Robot vacuum error codes (Roomba, Xiaomi, Roborock) and smart thermostat fault diagnostics.

Robot Vacuum Error Codes

Code Description Severity Affected Models Typical Cause Action
Error 1 Side Brush Stuck Warning Roomba 600/700/800/900, i series, Xiaomi Roborock S4/S5/S6/S7/S8 Hair (human or pet), string, or debris wrapped around side brush axle; brush motor over-current Power off robot. Lift side brush cover. Use scissors or seam-ripper to cut wrapped debris from brush hub. Re-insert brush. Verify brush spins freely by hand before resuming. Replace brush if bearing seized.
Error 2 Main Brush Stuck Warning Roomba i3/i4/i7/s9, Roborock S5 Max/S7/S8, Xiaomi Mi Robot 1S/2 Hair/fiber wrapped around main brush end caps; brush motor current exceed ~1.8A threshold Remove main brush module. Clean brush and both end caps. Use brush-cleaning tool to strip hair. Check motor connector. Replace main brush rubber every 3–6 months.
Error 3 Left Wheel Stuck Warning All Roomba/iRobot, Roborock, Xiaomi robot vacuums Hair/thread wrapped around left drive wheel axle; wheel encoder reports no motion; debris in wheel well Turn robot upside-down. Pull wheel module away from body. Cut wrapped debris. Manually rotate wheel. Re-assemble. If error persists, wheel motor assembly requires replacement.
Error 4 Right Wheel Stuck Warning All Roomba/iRobot, Roborock, Xiaomi robot vacuums Identical to Error 3 but on right drive wheel assembly Same procedure as Error 3, applied to right wheel. If both wheels report stuck, verify main logic board power (19V Roomba, 14.4V Roborock). Factory reset via companion app.
Error 5 Charging Error Warning Roomba i series, s series, Roborock E/S/Q series Dirty or oxidized charging contacts; dock not powered; battery below minimum charge voltage; faulty dock Clean both robot and dock contacts with isopropyl alcohol. Verify wall outlet powered. Check dock LED indicator. If dock is fine, suspect battery age — replace if > 2 years. Factory reset can clear comm glitches.
Error 6 Cliff Sensor Dirty Warning Roomba 600+, Roborock all models, Xiaomi Mi Robot Optical cliff sensor window blocked by dust/hair; sensor mis-calibrated; dark/black flooring false triggering Turn robot upside-down. Locate 4–6 cliff sensor optical windows. Clean with dry microfiber cloth (no liquid on sensors). If on black carpet, apply small white tape squares. Some models allow sensitivity adjustment via advanced app settings.
Error 7 Bin Full Warning Roomba i3+/i4+/i7+/s9+ (self-empty base), Roborock S7 MaxV, Q Revo Dust bin actually full; optical-bin-full sensor dirty; bin not properly seated; or base dust bag full Empty dust bin. Clean bin-full optical sensor pair inside bin compartment. Re-insert bin with clean contacts. On self-empty models, also verify base dust bag is not full.
Error 8 Vacuum Fan Motor Stalled Critical Roomba 900/i series, Roborock S5/S6/S7/S8 Main suction fan motor stalled; debris in fan impeller; motor bearing seized from dust ingress; driver FET failure Inspect suction fan intake port for debris blockage. If obstruction found, clear it. If no obstruction and error persists, fan motor is failed — replace full fan module. Check main PCB for burnt smell near motor driver on Roomba i series.

Smart Thermostat Error Codes

Code Description Severity Affected Models Typical Cause Action
E74 Rh Wire Report — Thermostat Wiring Warning Nest Thermostat E, Nest Learning Thermostat 2nd/3rd gen, Nest Thermostat 2020+ Rh wire voltage outside 24V expected range; loose connection; wire in wrong terminal; Rh/Rc confused Turn off HVAC breaker. Remove Nest from base. Check each wire in correct labeled terminal. Common mistake: Rh/Rc swapped, or jumper wire missing between Rh and Rc on single-transformer systems. Verify 24V AC at Rh-to-C. Use Nest Wiring Report in app.
E73 Rh Wire Disconnected Warning All Nest Thermostat models Rh (heat power) wire no longer making contact; HVAC transformer dead; blown HVAC fuse Check HVAC 3A or 5A low-voltage fuse on furnace control board. Check Rh wire at Nest terminal for loose fit. Verify 24V AC between Rh and C wire. Replace HVAC transformer if < 20V. Re-insert wire with terminal fully depressed before releasing.
E194 Wi-Fi / Cloud Connection Failure Warning Nest Thermostat, Nest Learning Thermostat, Ecobee 3/4/5 Lite/Premium Wi-Fi AP out of range; 2.4GHz band not available (Nest requires 2.4GHz); SSID changed; DHCP/DNS failure; Nest cloud outage Move router closer or install Wi-Fi extender. Confirm thermostat sees 2.4GHz SSID (NOT 5GHz-only). Re-enter Wi-Fi password through Settings > Network. Verify internet is up. Check Nest Service Status page. Factory reset only after all network checks pass.
H71 Heating Relay Fault Critical Nest Learning Thermostat 3rd gen, Nest Thermostat 2020+ Internal heating relay (W terminal) stuck ON or OFF; relay contact weld from high inrush; no C-wire causing brown-out Power off HVAC system immediately — stuck-on relay can overheat furnace. Check if C-wire is installed. If no C-wire, install C-wire adapter or Nest Power Connector. If C-wire present and code persists, thermostat relay is failed — replace unit. Verify furnace board not also damaged.
C155 Compressor Communication Fault Critical Nest Thermostat with heat pump systems, Ecobee with modulating equipment RS485 bus communication loss between thermostat and heat pump outdoor unit; proprietary Carrier Infinity/Bryant Evolution protocol NOT supported by Nest For Carrier/Bryant Infinity/Evolution: Nest does NOT support proprietary comm bus. Use manufacturer thermostat or rewire to standard R/W/Y/G/C. For standard heat pump, check reversing valve wire (O/B terminal) and 24V at outdoor unit. Check ACC terminal. Reset to factory defaults and re-walk equipment wizard.
